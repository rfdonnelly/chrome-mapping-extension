= Chrome Mapping Extension

This Chrome extension adds context menu shortcuts to open various location-based sites from Google Maps and CalTopo.

== Features

=== For Google Maps

* Open in CalTopo
* Open in Mountain Project
* NOAA forecast

=== For CalTopo

* Open in Google Maps
* Open in Mountain Project
